## 4. 执行第三轮 JMeter 压测

```bash

  -t STRESS-TEST/jmeter-260519/create-order-v4.jmx \
```

## 5. 压测后 Redis 状态确认

```bash
docker exec dismai-redis redis-cli HGETALL Dismai-d_mai_program_ticket_remain_number_hash_resolution_32_36
docker exec dismai-redis redis-cli HLEN Dismai-d_mai_program_seat_lock_resolution_hash_32_36
docker exec dismai-redis redis-cli HLEN Dismai-d_mai_program_seat_no_sold_resolution_hash_32_36
```

正常情况下，10000 条请求全部成功后：

- 余票 Redis hash 中票档 `36` 应接近或等于 `0`
- 已锁座位 hash 数量应接近或等于 `10000`
- 未售座位 hash 数量应接近或等于 `0`
